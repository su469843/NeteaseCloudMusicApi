📖 调用须知

  • 接口同时支持 GET/POST 请求，POST 请求需加时间戳避免缓存
  • 接口默认缓存 2 分钟，可在 URL 后加 ?timestamp=xxx 跳过缓存
  • 跨域请求需带上 withCredentials: true 或手动传入 cookie
  • 301 错误 = 未登录，503/460 错误 = IP 高频，加 realIP 参数可解决
  • 分页接口返回字段中有 more，为 true 表示还有下一页
  • 图片链接加 ?param=宽y高 可控制尺寸，如 ?param=200y200
  • 不需要 cookie 时可加 &noCookie=true
  • 可在 query 中加 proxy=your-proxy 使用代理
